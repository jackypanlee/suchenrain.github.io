`VS Studio Code`常用快捷键汇总，持续更新中...

<!--more-->

| 快捷键                                                 | &nbsp;                  |
| ------------------------------------------------------ | ----------------------- |
| `Alt` + <kbd>&uarr;</kbd>`/`<kbd>&darr;</kbd>          | 当前所在行 上/下移      |
| `Shift` +`Alt` + <kbd>&uarr;</kbd>`/`<kbd>&darr;</kbd> | 将当前行向上/下复制一行 |
| `Alt` + <kbd>&larr;</kbd>`/`<kbd>&rarr;</kbd>          | 光标历史 后退/前进      |
| `Ctrl` + `Enter`                                       | 另起一行                |
| `Ctrl` + `Shift` + `Enter`                             | 当前行之前插入新的一行  |
| `Ctrl` + `Shift` + `K`                                 | 删除当前行              |
